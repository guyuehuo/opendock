Welcome to OpenDock's documentation!
===================================
OpenDock is a versitile protein-ligand docking framework with diverse scoring functions and sampling strategies. 

Aim 
---
The project is intended to provide an open-source framework for protein-ligand docking by implementing customizable docking strategies.
Check out the :doc:`installation` section for installing, and the :doc:`docking_basic` section for further information.
